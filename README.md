# This project is created using React + Vite

// To create React App

1. Open cmd and enter

  "npm create vite@latest"
  Project name: ... halleyx
  Package name: ... halleyx
  Select a framework: >> React
  Select a variant: >>JavaScript

2. Install node modules

  "npm install"

3. Start the Project

  "npm run dev"
